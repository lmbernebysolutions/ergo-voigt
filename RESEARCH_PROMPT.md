# Research Prompt für Praxis-Präsentationsseite

Suche nach GitHub-Repositories für React/Next.js Komponenten, die mehrere Standorte oder Locations mit folgenden Features präsentieren:

**Anforderungen:**
- Präsentation von mehreren Standorten/Locations (3 Standorte)
- Jeder Standort benötigt eine Bildergalerie (Image Gallery)
- Jeder Standort benötigt eine Informationssektion mit Hauptdaten (Adresse, Kontakt, Öffnungszeiten, etc.)
- Moderne, responsive UI-Komponenten
- TypeScript Support bevorzugt
- Tailwind CSS kompatibel oder leicht anpassbar
- Next.js 15 App Router kompatibel

**Technologie-Stack:**
- React 18+
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (optional, für Animationen)

**Gesuchte Komponenten-Typen:**
- Location/Standort-Präsentationskomponenten
- Multi-Location Showcase Components
- Location Cards mit Galerie
- Tabbed oder Accordion-basierte Location-Präsentationen
- Image Gallery Komponenten für Standorte

**Wichtige Kriterien:**
- Aktive Repositories (letztes Update in den letzten 12 Monaten)
- Gute Dokumentation
- TypeScript Support
- Accessibility-Features (ARIA, Keyboard Navigation)
- Mobile-responsive Design
- Performance-optimiert (Image Optimization, Lazy Loading)

**NICHT gesucht:**
- Maps/Geolocation Libraries
- Booking/Appointment Systems
- E-Commerce Location Features
- Backend/API Implementierungen

Bitte liste die Top 5-10 Repositories mit kurzer Beschreibung, Sternen, letztem Update und Hauptfeatures auf.
